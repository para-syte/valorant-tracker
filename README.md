# Valorant Tracker

Basic application and personal project used to retrieve valorant players' statistics.

## How to run

run the development server:

```bash
cd valorant-tracker
npm run dev
```
Open http://localhost:5173/

## How it works
Retrieves basic valorant players' statistics such as name, tag, account level, rank, and more.

## Tech used

- React
- Js/CSS
- henrikdev unofficial Valorant API
